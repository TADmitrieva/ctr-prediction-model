# ctr-prediction-model
ML-модель для предсказания вероятности клика (CTR) в рекламных логах. Логистическая регрессия, отбор признаков (RFE) и калибровка вероятностей.

# Прогнозирование CTR рекламных объявлений

Данный проект посвящен разработке модели машинного обучения для предсказания вероятности клика (CTR) на рекламные баннеры. В ходе работы была построена стабильная и интерпретируемая модель на основе логистической регрессии.

## Основные результаты
* **Итоговая метрика PR-AUC:** 0.3649 (в 2.1 раза выше базовой модели с PR AUC = 0.1720).
* **Модель:** Logistic Regression с оптимальной регуляризацией (C=0.1).
* **Оптимизация:** Признаковое пространство сокращено с 115 до 10 ключевых параметров.

## Стек технологий
* **Python 3.12+**
* **Scikit-learn** (Pipeline, GridSearchCV, TargetEncoder)
* **Phik** (анализ нелинейных корреляций)
* **Matplotlib & Seaborn** (визуализация)

## Алгоритм
Logistic Regression с L2-регуляризацией (C=0.1) и калибровкой вероятностей.

## Ключевые инсайты
1. Ключевыми предикторами клика являются идентификатор рекламной кампании (`C14`) и площадка размещения (`site_id`).
2. Модель обладает отличной калибровкой (ECE < 2%), что позволяет использовать её вероятности для расчета eCPM в реальном времени.

## Структура репозитория
ctr_prediction.ipynb — основной файл с полным циклом исследования: EDA, отбор признаков (RFE), обучение и калибровка.
final_model.joblib — финальная обученная и откалиброванная модель (включает в себя Pipeline с препроцессором).
selected_features_info.joblib — метаданные об отобранных признаках.
requirements.txt — список зависимостей для воспроизведения среды.

## Особенности данных
Данные представляют собой анонимизированные логи рекламных аукционов. В ходе проекта реализован сложный пайплайн обработки:
* Target Encoding для признаков с высокой кардинальностью.
* One-Hot Encoding с защитой от неизвестных категорий и мультиколлинеарности.
* Feature Selection: сокращение признакового пространства с 115 до 10 наиболее информативных параметров.
  
## Рекомендации для бизнеса
Основным вектором оптимизации рекламного бюджета является фокус на высокодоходных площадках (site_id) и конкретных конфигурациях объявлений (C14, C17). Модель позволяет в реальном времени отсекать показы с низкой вероятностью клика, что значительно снижает стоимость привлечения пользователя (CPA) и повышает общую рентабельность кампаний (ROAS).
